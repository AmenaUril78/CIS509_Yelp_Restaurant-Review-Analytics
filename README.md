# 🍽️ Yelp Restaurant Performance Analyzer

CIS 509 – Unstructured Data Analytics  
Team 8

## Overview

This project transforms unstructured Yelp review data into actionable operational insights for independent restaurants.

Independent restaurants often lack structured analytics tools. This application applies Natural Language Processing (NLP) techniques to:

- Detect sentiment trends
- Identify dominant complaint themes
- Generate prioritized operational recommendations

## Features

- Sentiment Analysis (TextBlob)
- TF-IDF Keyword Extraction
- Interactive Dashboard
- Operational Action Plan Generator
- Multi-page Streamlit Interface
